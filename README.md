# Off-Path Bitcoin Hijacking via Wi-Fi MAC-Layer Side Channels

This repository contains the **research artifacts** for the paper:

> **Off-Path Bitcoin Hijacking via MAC-Layer Side Channels in Wi-Fi Networks**  
> *(submitted for double-blind review)*

The code and scripts in this repository support the **reproducibility of the experimental results** reported in the paper, including Bitcoin traffic identification, TCP state inference, and Bitcoin protocol–level manipulation enabled by Wi-Fi side channels.
